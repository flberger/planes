Class Hierarchy

  • clickndrag.Plane - A Plane is a surface in a hierarchy of surfaces.
    • clickndrag.Display - Click'n'Drag main screen class.
    • clickndrag.gui.Container - A Container for Planes.
      • clickndrag.gui.GetStringDialog - A combination of Container, Label, TextBox and Button that asks the user for a string.
      • clickndrag.gui.OkBox - A box which displays a message and an OK button.
      • clickndrag.gui.OptionList - A list of options to select from.
      • clickndrag.gui.OptionSelector - An OptionSelector wraps an OptionList and an OK button, calling a callback when a selection is confirmed.
    • clickndrag.gui.Label - A clickndrag.Plane which displays a text.
      • clickndrag.gui.Button - A clickndrag plane which displays a text and reacts on mouse clicks.
      • clickndrag.gui.Option - A subclass of Label which handles mouseclicks, to be used in an OptionList.
      • clickndrag.gui.OutlinedText - A Label with outlined text and a transparent background.
      • clickndrag.gui.TextBox - A box where the user can type text.
    • clickndrag.gui.ScrollingPlane - This class implements a fixed-dimension plane with a scroll bar to scroll its content plane.
